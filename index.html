Popovの超安定論

Popovの超安定論は図に示す線形定常ブロックと非線形時変ブロックで構成されるフィードバックシステムが次の二つの条件を満足するとき、そのフィードバックシステムは安定(超安定)であることを示しています。

・線形ブロックの伝達関数が強正実条件を満足する。

・Popovの積分不等式が成立する。

強正実条件:

線形ブロックの伝達関数\(G(s)\)が次の条件を満足することです。(\(s\)はラプラス変数です。)

(1)\(G(s)\)が実のsに対して実である。

(2)\(G(s)\)が漸近的に安定、すなわちs面の右半平面\(Re[s]\ge0\)に極がない。(\(Re[*]\)は\(*\)の実数部分です。)

(3)\(-\infty < \omega < \infty\)のすべての\(\omega\)に対して\(Re[G(j\omega)]\ge0\)。 (\(j\)は虚数単位、\(\omega\)は角振動数です。)

Popovの積分不等式:

線形ブロックの出力を\(v\)、非線形ブロックの出力を\(w\)として、Popovの積分不等式は次式で与えられます。

\begin{align} \int_{0}^{t}vw dt \ge -\gamma_0^2 \end{align}

ここに、\(\gamma_0\)は定数です。

それでは、具体的なフィードバックシステムについて考察してみましょう。

ここでは入力\(u\)、出力\(x\)とし、調整可能な定数\(\hat b(> 0)\)含む下記の実システムを想定します。

\begin{align} \ddot x + a_1\dot x + a_2 x = \hat{b}u \label{eq:RealSys} \end{align} それでは、超安定論に従って定数\(b\)の調整則を決定しましょう。 まず、このシステムに対して下記の規範モデルを与えます。 \begin{align} \ddot x_M + a_1\dot x_M + a_2 x_M = bu \label{eq:RefMdl} \end{align}

ここに、\(x_M\)は規範モデルの出力、\(b (> 0)\)は定数です。 そこで、実システムの\(\hat b\)を調整して、規範モデルの係数\(\hat b\)と一致させる方法を見つけましょう。 両者が一致すれば、\(x\)は\(x_M\)と一致します。

まず、式\eqref{eq:RefMdl}の両辺から式\eqref{eq:RealSys}の両辺をそれぞれ引きます。

\begin{align*} (\ddot x_M - \ddot x) + a_1(\dot x_M - \dot x) + a_2(x_M - x) = (\hat b - b)u \end{align*}

次に\(e=\dot x_M - \dot x\)、\(\tilde b =\hat b - b\)とおくと、次式になります。

\begin{align} \ddot e + a_1\dot e + a_2 e = -\tilde b u \label{eq:ErrSys} \end{align}

式\eqref{eq:ErrSys}の左辺を定常線形ブロック、右辺を非線形時変ブロックに分け、フィードバックシステムを構成します。

\(-w\)を入力、\(v\)を出力とする線形定常ブロック：

\begin{align} \ddot e + a_1\dot e + a_2 e &= -w\\ v &= e + c\dot{e} \end{align}

\(v\)を入力、\(w\)を出力とする非線形時変ブロック：

\begin{align} w = \tilde b(v) u \end{align}

これで、Popovの超安定論を適用するための線形定常ブロックと線形定常ブロックを構成し、 互いの入出力関係を確認できました。後(式\eqref{eq:ModPara})で分かりますが\(\tilde b\)は\(v\)に依存するので、 \(\tilde{b} = \tilde{b}(v)\)と表記しています。 まず、Popovの積分不等式を満足させましょう。

Popov積分:

\begin{align} \int_{0}^{t}vw dt = \int_{0}^{t}\tilde b u v dt \end{align}

Popovの積分不等式を満足させるために次式を与えます。

\begin{align} \dot{\tilde b}(v) = \alpha u v \label{eq:ModPara} \end{align}

ここに、\(\alpha\)は定数です。すると、Popovの積分不等式が満足されることを次式で確認できます。

\begin{align*} \int_{0}^{t}vw dt = \frac{1}{\alpha}\int_{0}^{t}\dot{\tilde b} \tilde b dt = \frac{1}{2\alpha}[\tilde b^2]_0^t \ge -\frac{1}{2\alpha}\tilde b^2|_{t=0} \end{align*}

次に強正実条件を確認します。 \(v\)、\(w\)のラプラス変換をそれぞれ\(V(s)\)、\(W(s)\)とすると、 \(v\)から\(w\)への伝達関数は次式となります。

\begin{align} G(s) = -\frac{V(s)}{W(s)} = \frac{1 + cs}{s^2 + a_1 s +a_2} \label{eq:TrnsFnc} \end{align}

実数のsに対してG(s)は実数となり、極(分母：\(s^2 + a_1 s +a_2=0\)の解)の実数部分は\(-a_2/2\)なのでs面の右半平面\(Re[s]\ge0\)に極はありません。 したがって、強正実条件(1)、(2)を満足します。 式\eqref{eq:TrnsFnc}の\(s\)に\(j\omega \)を代入します。

\begin{align*} Re[G(j\omega)] = Re[\frac{1+jc\omega}{a_2 - \omega^2 + ja_1 \omega}] = Re[\frac{(1+jc\omega)\{(a_2 - \omega^2) - ja_1 \omega\}}{(a_2 - \omega^2)^2 + (a_1\omega)^2}] = \frac{a_2 + (a_1c -1)\omega^2}{(a_2 - \omega^2)^2 + (a_1\omega)^2} \end{align*}

\(a_1c -1 > 0\)であれば\(Re[G(j\omega)] > 0\)となり、強正実条件(3)が確認できます。

\(b\)が定数であること(\(\dot b = 0\))に注意すれば、式\eqref{eq:ModPara}の\(\dot{\tilde b}=\dot{\hat b}\)です。 そこで、次式が成立するように実システムの\(\hat b\)を調整します。

\begin{align} \dot{\hat b} = \alpha u v \end{align}

これで、Popovの超安定条件が満足されるので次式が成立します。

\begin{align*} \lim_{t \rightarrow \infty} v = 0,~~~\lim_{t \rightarrow \infty} \hat b = b. \end{align*}

こうして実システムは漸近的に規範モデルと一致します。 モデル規範適応制御ではより具体的に説明します。
